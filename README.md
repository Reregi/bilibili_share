# chrome-extension——bilibili_share 
# 哔哩哔哩视频页分享-chrome扩展程序
**解决摸鱼的适合看b站分享给好友时，直接发链接没有视频简介的尴尬问题😅**

## 使用方法
1. 打开chrome扩展
2. 加载解压扩展程序
3. 打开b站视频页
4. 点击获取
5. 点击复制
6. 将视频简介图片复制到你想复制的地方


## 使用依赖
* [QRCode.js](https://github.com/davidshimjs/qrcodejs)
* [html2canvas.js](http://html2canvas.hertzen.com/)
